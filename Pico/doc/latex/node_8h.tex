\subsubsection[{pack\_\-nodes}]{\setlength{\rightskip}{0pt plus 5cm}int pack\_\-nodes ({\bf Node} $\ast$$\ast$$\ast$ {\em array\_\-of\_\-nodes}, \/  int {\em cur\_\-size}, \/  {\bf Node} $\ast$ {\em n})}}
\label{node_8h_a1e1a21ca62ad4d35baa0bf0694e5a241}
Pushes 'n' Nodes on bottom of an array of Node$\ast$ which contains originally 'cur\_\-size' entries. Returns the new number of entries (ie 'cur\_\-size+1') in the array 'array\_\-of\_\-nodes'. This function is convenient to be used with create\_\-node (see its last argument). On the first call, 'array\_\-of\_\-nodes' should not be allocated, and 'cur\_\-size' should be zero. Implementation limit: since these Node$\ast$ structures are meant to be used in a compiler, one expects an upper limit of MAX\_\-CHILDREN\_\-NUMBER nodes to be packed. Trying to pack more than this limit must raise an error. Typical use: Node$\ast$$\ast$ children; pf1 = create\_\-leaf(1, int\_\-node, \char`\"{}1\char`\"{}, NULL); pf3 = create\_\-leaf(1, int\_\-node, \char`\"{}2\char`\"{}, NULL); pack\_\-nodes(\&children, 0, pf1); pack\_\-nodes(\&children, 1, pf3); 
\begin{DoxyParams}{Parâmetros}
\item[{\em array\_\-of\_\-nodes}]the array where the node will be pushed \item[{\em cur\_\-size}]the current size of the array \item[{\em n}]the node to be pushed \end{DoxyParams}
\begin{DoxyReturn}{Retorna}
the new number of entries (cursize + 1) 
\end{DoxyReturn}
\begin{DoxyWarning}{Aviso}
cur\_\-size should be less than MAX\_\-CHILDREN\_\-NUMBER 
\end{DoxyWarning}
